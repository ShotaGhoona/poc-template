# POC開発ユースケース一覧

Lovableが生成する2つのプロジェクト（archaive-v4 / elegant-workflow-craft）を分析し、POC開発で頻出するユースケースを網羅的に列挙する。

## 1. 認証・認可

1. **メール/パスワード認証を導入したい**
2. **ソーシャルログイン（Google/GitHub）を追加したい**
3. **パスワードリセット機能を実装したい**
4. **サインアップ後に自動でユーザープロフィールを作りたい**
5. **ログイン済みユーザーだけがアクセスできるページを作りたい（Protected Route）**
6. **未ログインユーザーをログインページにリダイレクトしたい**
7. **ログイン中のユーザーがAuthページにアクセスしたらダッシュボードにリダイレクトしたい**
8. **ユーザーのロール（admin/member）に応じてUIを出し分けたい**

10. **初回ログイン時にオンボーディングフローを挟みたい**
